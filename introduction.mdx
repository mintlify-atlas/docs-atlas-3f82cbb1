## What is CodesEverywhere?

CodesEverywhere is an open-source code snippet sharing platform that lets developers share their code with the world — or keep it private. Built with a Spring Boot backend and React frontend, it provides a simple, fast way to store and discover code snippets across any programming language.

## Key features

- **Public feed** — Browse randomly ordered public snippets from the community dashboard
- **Code sharing** — Upload snippets with a title, language/type label, description, and public or private visibility
- **Authentication** — A combined login and signup flow: new accounts are created automatically on first use
- **User posts** — Authenticate to view all your own snippets in one place
- **Infinite scroll** — The dashboard loads more snippets automatically as you scroll
- **REST API** — A clean Spring Boot API with three endpoints covering all core functionality
